Secrets.

Declare application secrets with @KoSecret. Resolved from environment variables locally, from secret managers in production.

Using secrets Java 
@KoService("payment-service")
public class PaymentService {

    @KoSecret("stripe-api-key")
    private KoSecret stripeKey;

    @KoSecret("webhook-signing-secret")
    private KoSecret webhookSecret;

    @KoAPI(method = "POST", path = "/payments", auth = true)
    public PaymentResponse charge(ChargeRequest req) {
        Stripe.init(stripeKey.value());
        // ...
    }
}
Resolution Order

Secrets are resolved based on the environment: environment variables for local development, AWS Secrets Manager, GCP Secret Manager, or HashiCorp Vault in production — configured via infra-config.json.
