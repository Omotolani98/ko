Annotation Reference.

Complete reference for all Kọ́ annotations.

AnnotationTargetPurposeKey Attributes
@KoServiceTypeDeclares a microservice boundaryvalue (name)
@KoAPIMethodDeclares an HTTP endpointmethod, path, auth, permissions, expose
@KoDatabaseFieldDeclares a SQL databasename, migrations
@KoPubSubFieldDeclares a message topictopic, delivery
@KoSubscribeMethodSubscribes to a topictopic, name, maxRetries, deadLetter
@KoCacheFieldDeclares a key-value cachename, keyType, ttl
@KoCronMethodDeclares a scheduled jobschedule, name
@KoBucketFieldDeclares object storagename, publicRead
@KoSecretFieldDeclares an application secretvalue (name)
@KoServiceClientFieldInjects a generated service clientvalue (service name)
@PathParamParameterBinds a path parametervalue (param name)

Retention Policy

All Kọ́ annotations use @Retention(RUNTIME) so they are available both at compile time (for the annotation processor) and at runtime (for the Spring Boot auto-configuration engine).
