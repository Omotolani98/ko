>_ Reference

CLI Reference.

The ko command-line tool orchestrates local development, builds, and code generation.

Terminal Shell 
# Start local dev environment
$ ko run

# Build production Docker image
$ ko build docker --config infra-prod.json my-app:latest

# Generate TypeScript API client
$ ko gen client --lang typescript --output ./frontend/src/api

# Run tests with isolated infrastructure
$ ko test

# View the application model
$ ko model

# Open the dev dashboard
$ ko dashboard

Commands

CommandDescription
ko runStart the application with auto-provisioned local infrastructure
ko buildBuild the application (compile + generate model)
ko build dockerBuild a production Docker image
ko testRun tests with isolated Testcontainer infrastructure
ko gen clientGenerate typed API clients (TypeScript, etc.)
ko modelDisplay the Application Model
ko dashboardOpen the dev dashboard in a browser
ko versionPrint version information
